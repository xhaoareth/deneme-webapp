# Deneme Webapp

Modern, responsive single-page app built with Vite + React. Includes a todo list, quick notes, and a light/dark theme toggle with persistence via `localStorage`.

## Getting started

```bash
npm install
npm run dev
```

> If your environment blocks access to the public npm registry, run the commands from a network with registry access (or via a configured proxy) and then open the local dev server URL from the console (default: http://localhost:5173).

## Features
- Add/remove todos and notes
- Theme toggle with persisted preference
- Data stored in `localStorage`
- Responsive layout with modern styling
